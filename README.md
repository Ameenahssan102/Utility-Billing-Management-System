# Utility Billing Management System

## Overview
The Utility Billing Management System is a comprehensive application designed by Ameen Ahssan to manage utility bills for residential and commercial customers. It provides functionalities for customer registration, bill generation, bill viewing, and maintains a comprehensive bill history for each customer.

## Features
- **Customer Registration:** Allows new customers to register by providing essential details such as name, address, and contact information.
- **Bill Generation:** Calculates utility bills accurately based on gas and electricity readings, tariffs, and usage.
- **Bill Viewing:** Enables customers to conveniently view and download their bills for transparency and record-keeping.
- **Bill History:** Maintains a detailed bill history for each customer, allowing them to access and review past bills.

## Technologies Used
- JavaFX for the frontend user interface.
- Java for backend logic and calculations.
- File handling for data storage and retrieval.
- Git for version control and collaboration.
- GitHub for hosting the project repository.

## Installation
1. Clone the repository to your local machine:
2. Import the project into your preferred Java IDE.
3. Run the `Main.java` file to launch the application.

## Usage
1. Launch the application.
2. Register as a new customer or log in with existing credentials.
3. Generate utility bills based on gas and electricity readings.
4. View and download bills for transparency and record-keeping.
5. Access the bill history to review past bills.
